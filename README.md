# 아티스트 포트폴리오 사이트 관리 가이드

이 웹사이트는 코딩 지식 없이도 **엑셀(Excel) 데이터를 복사하여 붙여넣는 방식**으로 모든 내용을 관리할 수 있습니다. `data.js` 파일 하나만 수정하면 됩니다.

---

## 🚀 사이트 데이터 수정 방법 (data.js)

`data.js` 파일을 텍스트 에디터(메모장, VS Code 등)로 엽니다. 모든 데이터는 백틱(`` ` ``) 기호 사이에 입력해야 합니다.

### 🎨 1. 기본 정보 설정 (SITE_CONFIG)

- `title`: 브라우저 탭 이름
- `artistName`: 사이트 상단 작가명
- `email`: 문의 받을 메일 주소

### 🖼️ 2. 작품 데이터 관리 (WORKS_CSV)

엑셀에서 관리하는 작품 정보를 그대로 붙여넣습니다. (쉼표나 탭으로 구분)

- **열 순서**: 작품제목, 크기, 재료, 연도, 이미지파일명
- **예시**: `꽃의 기록, 100x100cm, Oil on canvas, 2023, work1.jpg`
- 이미지파일명은 `images/` 폴더 안의 파일명을 토대로 입력하세요.

### 📜 3. CV 데이터 관리 (EDUCATION, EXHIBITIONS, AWARDS)

학력, 전시, 수상 경력을 각각의 CSV 섹션에 입력합니다.

- **방법**: 한 줄에 한 항목씩 입력하세요. 엑셀에서 한 열을 복사해서 붙여넣으면 됩니다.

---

## 📁 이미지 파일 관리

1. 웹사이트 폴더 내의 `images` 폴더 안에 작품 이미지를 넣습니다.
2. `data.js`의 이미지파일명과 실제 폴더 내의 파일명이 일치해야 합니다. (대소문자 주의)
3. 이미지 권장 크기: 웹 로딩 속도를 위해 파일당 1MB 이하를 권장합니다.

---

## 🛠️ 메뉴 이름 수정 (index.html)

메뉴 이름을 바꾸고 싶을 때만 `index.html`을 엽니다.

1. `<nav>` 태그 안의 `Home`, `Works`, `CV`, `Contact` 글자만 바꿉니다.
2. **주의**: `data-section` 속성이나 `id` 값은 절대 수정하지 마세요.

---

## 🔄 사이트 반영

수정한 파일을 **저장**하고 브라우저에서 사이트를 **새로고침**하면 즉시 반영됩니다.
